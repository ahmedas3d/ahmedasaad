# Ahmed Asaad - Flutter Developer Portfolio

A professional, tech-focused portfolio website for Ahmed Asaad, a Junior Flutter Mobile App Developer with 1+ years of experience. Built with React, TypeScript, and Tailwind CSS.

## 🚀 Features

- **Terminal-Inspired Design**: Tech-focused aesthetic with terminal elements and code styling
- **Responsive Layout**: Mobile-first approach optimized for all devices
- **Dark/Light Mode**: Smooth theme switching with system preference detection
- **Dramatic Animations**: Framer Motion powered terminal boot sequence and interactions
- **Modern Tech Stack**: React 18, TypeScript, Tailwind CSS
- **5 Featured Projects**: Comprehensive showcase of mobile applications
- **SEO Optimized**: Meta tags and structured data for search visibility

## 📱 Projects Showcased

1. **Sportify** - Football app with live scores and match schedules
2. **Home Food** - Platform connecting customers with home-cooked meals
3. **News Hub** - Real-time news aggregation with clean architecture
4. **Scholar Chat** - Real-time messaging platform
5. **Flowery** - E-commerce platform for floral products

## 📦 Installation & Setup

1. **Clone the repository**

```bash
git clone <repository-url>
cd ahmed-portfolio
```

2. **Install dependencies**

```bash
npm install
```

3. **Start development server**

```bash
npm start
```

4. **Build for production**

```bash
npm run build
```

## 🔧 Customization

To customize this portfolio for your own use:

1. **Update Personal Data**: Modify `src/data/personal.ts`
2. **Update Projects**: Edit `src/data/projects.ts`
3. **Update Skills**: Modify `src/data/skills.ts`
4. **Change Colors**: Update `tailwind.config.js`
5. **Update Content**: Edit page components in `src/pages/`
